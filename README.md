# Book-Wurm
Collection of online books and PDFS. 

add pdf's to the pdf folder
add cover images to the images folder

in terminal run 

node build.js

this will cross check the pdf folder with the images folder for likeness in the titles and pair them to link in the listing.
once the netWork.html is rebuilt your good to go and add another pdf and image or add a shyt load if ya want. 

simple resource viewer. feel free to customize anything, just clone it if ya do or contribute to build this one. -NET
